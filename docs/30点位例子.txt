1.	第18章 五 焰色反应
实验原理：
金属原子受热时，外层价电子（ns/np）吸收热能，从基态跃迁至激发态；随即以辐射方式跳回低能级，释放特征频率光子（ΔE = hν）。因各元素原子能级差固定，故发射光谱具有唯一性，呈现特征焰色。钾的紫光需通过钴玻璃滤除钠黄光干扰。

现象解释：
各离子焰色（实验记录参考）：
•	Li⁺：洋红色（紫红）
•	Na⁺：黄色（持久明亮）
•	K⁺：紫色（需透过钴玻璃观察）
•	Ca²⁺：砖红色（橙红）
•	Sr²⁺：猩红色（洋红）
•	Ba²⁺：黄绿色（苹果绿）

安全提示：
1.	防烫伤：镍丝在火焰中灼烧后温度极高，冷却前勿触碰或接触点滴板。
2.	防腐蚀：蘸取浓盐酸时需在通风处操作，避免皮肤接触（盐酸有腐蚀性）。
3.	防火源：蘸取溶液前确保镍丝已冷却，否则高温遇酒精蒸气可能引燃（若使用酒精灯）。
4.	废液处理：实验后的含金属离子废液应回收，勿随意倾倒。

2.	第18章 一 1.钠加热燃烧实验
实验原理：
2 Na + O2 = Na2O2
2 Na2O2 +2 H2O = 4 NaOH + O2↑
Na2O2 + 2 H2O = 2 NaOH + H2O2↑ 
2 H2O2 = 2 H2O + O2↑
2 KMnO4 + 5 H2O2 + 3 H2SO4 = K2SO4 + 2 MnSO4 + 5 O2↑+ 8 H2O

现象解释：
1.钠燃烧：钠受热先熔化成银白色小球，然后剧烈燃烧，发出黄色火焰，生成淡黄色固体
2.加水：淡黄色固体遇水剧烈反应，放出大量无色无味气体（氧气），溶液呈强碱性（pH 试 纸变蓝或酚酞变红）。
3.酸化后加 KMnO₄：紫色高锰酸钾溶液迅速褪为无色（或浅粉色），同时可能产生少量气泡。 

安全提示:
1.	防燃烧爆炸：钠燃烧时火焰剧烈，切勿近距离直视，蒸发皿下方需垫石棉网，周围移除所有易燃物。
2.	防腐蚀灼伤：过氧化钠（Na₂O₂）具强氧化性和腐蚀性，切勿用手直接接触产物；用玻璃棒捣碎时需戴护目镜，防碎屑飞溅入眼。
3.	防喷溅：产物溶于水时放热剧烈，试管口切勿对人，应缓慢加水并冷却，防止碱性溶液喷溅。
4.	酸液安全：使用6 mol/L H₂SO₄时需戴手套，避免皮肤接触（浓硫酸稀释错误操作会放热，但本实验仅为酸化，仍需谨慎）。KMnO₄溶液沾到皮肤可用大量水冲洗。

3.	第18章 一 2.镁条燃烧实验
实验原理：
2Mg + O₂ → 2MgO
3Mg + N₂ → Mg₃N₂
2Mg + CO₂ → 2MgO + C
Mg3N2 + 6 H2O = 3 Mg(OH)2 + 2 NH3↑

现象解释：
1.镁条剧烈燃烧，发出耀眼的白光，放出大量热。
2.生成白色固体（主要产物为 MgO），同时伴有少量白烟（MgO 的微小颗粒）。 
3.若在空气中燃烧，产物中还可能含有少量淡黄色的氮化镁（Mg3N2），以及少量黑色物质（碳单质）。 
4.燃烧后的白色固体落入蒸发皿中，质地疏松。

安全提示：
1.	防强光伤害：镁燃烧发出强烈白光并含紫外线，严禁肉眼直接长时间注视，应佩戴墨镜或护目镜，或侧面观察。
2.	防高温灼伤：燃烧时温度极高（约3000°C），镁条下方必须垫石棉网或沙土，防止烧坏桌面；熔化的氧化镁会滴落，注意远离易燃物。
3.	防错误灭火：绝对不能用CO₂灭火器，因镁会与CO₂反应（2Mg + CO₂ → 2MgO + C），应用干沙覆盖灭火。
4.	操作规范：用坩埚钳夹紧镁条末端，手臂勿伸入火焰正上方，身体与火焰保持安全距离。
4.第18章 一 3.钠与水反应
实验原理：
2 Na + 2 H2O = 2 NaOH + H2↑

现象解释：
1.钠浮在水面上（密度小于水），熔化成银白色小球（反应放热，钠熔点低）。
2.小球在水面上迅速游动，发出“嘶嘶”声，并逐渐变小直至消失。
3.有无色无味气体放出（氢气）。
4.滴加酚酞，溶液变红（生成 NaOH，溶液呈强碱性）。
5.递变规律:同一主族从上到下(Na→K→Rb→Cs)金属性增强，与水反应剧烈程度增加；同周期Na比Mg 活泼， Mg比AI活泼。碱土金属中Ca比Mg活泼，但Ca与水反应仍不如Na剧烈，因Na的金属性更强。

安全提示：
1.	防爆炸：钠与水反应剧烈放热并产生H₂，H₂与空气混合遇明火易爆炸，烧杯口应用漏斗盖住，远离火源。 
2.	防腐蚀灼伤：反应后溶液含强碱NaOH，具有腐蚀性，避免皮肤接触；若不慎沾到，立即用大量水冲洗。 
3.	防碎片飞溅：钠反应可能剧烈到使小块金属弹跳或飞溅，应保持观察距离，必要时戴护目镜。 
4.	用剩处理：未反应的钠绝对不可丢入水槽或垃圾桶，应放回煤油中保存。 

5.第18章 一 5镁与冷／热水反应
实验原理：
Mg + 2 H2O = Mg(OH)2 + H2↑

现象解释：
1.	冷水：反应极慢，几乎观察不到气泡，数分钟后可能有极少量气泡附着在镁条表面，溶液 仍为中性（滴加酚酞不变红）。 
2.	热水：镁条表面迅速产生大量无色气泡（氢气），溶液逐渐变浑浊（生成 Mg(OH)₂沉淀）， 滴加酚酞变红（溶液呈弱碱性）。
3.	镁与水反应生成的 Mg(OH)2 难溶于水，覆盖在镁表面阻碍反应；加热时 Mg(OH)2 溶解度增 大且反应速率加快。
4.	递变规律:同一主族从上到下(Na→K→Rb→Cs)金属性增强，与水反应剧烈程度增加；同周期Nal比Mg 活泼， Mg比AI活泼。碱土金属中Ca比Mg活泼，但Ca与水反应仍不如Na剧烈，因Na的金属性更强。

安全提示：
1.	防烫伤：热水试管温度较高，用试管夹夹持，勿直接触碰。
2.	防火防爆：反应释放H₂，远离明火，试管口勿对人。
3.	防划伤：打磨镁条时注意砂纸或镁条边缘划伤手指。

6. 第18章 一 6.钙与水反应
实验原理：
Ca + 2 H2O = Ca(OH)2 + H2↑

现象解释：
1.	钙块沉入水底（密度大于水，但较钠略大），反应放出无色气泡（氢气），钙块表面产生大 量气泡，逐渐溶解。
2.	溶液变浑浊（生成 Ca(OH)₂微溶于水），滴加酚酞变红（溶液呈碱性）。
3.	反应比钠与水的反应稍缓和（钠更剧烈），但明显比镁剧烈。

安全提示：
1.	防腐蚀灼伤：反应生成Ca(OH)₂（强碱性），具有腐蚀性，避免皮肤接触；若不慎沾到，立即用大量水冲洗。
2.	防火防爆：反应释放H₂，遇明火易爆燃，试管口远离火源，勿对准他人。
3.	防试管破裂：钙与水反应放热，试管外壁可能变烫，勿用手直接握持，可用试管夹夹持。

7. 第15章 二 （1）亚硝酸的生成与分解
   实验原理：
2NaNO₂ + H₂SO₄ → 2HNO₂ + Na₂SO₄
2HNO₂ → N₂O₃ + H₂O
2N₂O₃ → NO↑ + NO₂↑
2HNO₂ → NO↑ + NO₂↑ + H₂O
3NO₂ + H₂O → NO↑ + 2HNO₃

   现象解释：
将冷冻的饱和NaNO₂溶液与稀H₂SO₄混合后，溶液由无色变为蓝色（生成亚硝酸，N₂O₃使溶液呈淡蓝色），同时产生红棕色气体（NO₂）。放置一段时间后，蓝色逐渐褪去（亚硝酸完全分解），溶液变为无色（最终生成稀硝酸），红棕色气体也慢慢消失（NO₂与水反应生成HNO₃和NO）。

安全提示：
1.	防中毒：反应产生NO和NO₂均为有毒气体，实验必须在通风橱中进行，避免吸入。
2.	防腐蚀：H₂SO₄和生成的HNO₃具有腐蚀性，避免皮肤接触；若不慎沾到，立即用大量水冲洗。
3.	低温操作：NaNO₂溶液需提前用冰水冷冻，操作时注意手部防冻（可用夹子取放试管）。
4.	废液处理：反应后废液含亚硝酸盐和硝酸盐，具毒性，应回收处理，勿随意倾倒。

8. 第15章 二 （2）亚硝酸的氧化性
  实验原理：
2NO₂⁻ + 2I⁻ + 4H⁺ → 2NO↑ + I₂↓ + 2H₂O
2NO + O₂ → 2NO₂
2HNO₂ → NO↑ + NO₂↑ + H₂O

现象解释：
1.常温：溶液立即由无色变为棕黄色（生成I₂），试管口有红棕色烟雾（NO遇O₂氧化为NO₂）。
2.加热：棕黄色加深（升温加快反应，生成更多I₂），随后颜色变浅（I₂受热升华逸出），试管口出现紫红色蒸气（I₂升华），同时伴有红棕色气体（HNO₂分解产生NO₂）。

安全提示：
1.	防中毒：反应产生NO和NO₂均为有毒气体，必须在通风橱中操作，避免吸入。
2.	防腐蚀：H₂SO₄具有腐蚀性，避免皮肤接触。
3.	防烫伤：微热试管时用试管夹夹持，勿直接用手握持加热部分。
4.	废液处理：含I₂和亚硝酸盐的废液具毒性，应回收处理，勿随意倾倒。

9. 第15章 二 （3）亚硝酸的还原性

2.	废液处理：含KMnO₄和亚硝酸盐的废液应回收处理，勿随意倾倒。

14. 第15章 三 （1）3.浓硝酸/稀硝酸 + 铜
实验原理：
Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
2NO + O₂ → 2NO₂

现象解释：
1.	浓硝酸：铜片溶解，溶液变为蓝绿色（生成Cu²⁺），产生大量红棕色气体（NO₂），溶液可能因NO₂溶解而呈黄绿色。
2.	稀硝酸：铜片溶解，溶液变为蓝色（生成Cu²⁺），产生无色气体（NO），在试管口遇空气变为红棕色（NO被O₂氧化为NO₂）。
3.	硝酸浓度越高，氧化性越强，但还原产物中氮的氧化数降低较少（主要生成 NO₂）；浓度越 低，氧化性减弱，但还原产物中氮的氧化数降低更多（生成 NO、N₂O 甚至 NH₄⁺）。这是因 为稀硝酸有大量水，有利于氮的深度还原。

安全提示：
1.	防中毒：反应产生NO₂有毒气体，在通风橱中操作。
2.	防腐蚀：浓硝酸具有强腐蚀性，避免皮肤接触。
3.	废液处理：含Cu²⁺的废液应回收处理，勿随意倾倒。

15. 第15章 三 （3）2. FeSO₄·7H₂O + NaNO₃ + 浓硫酸
实验原理：
3Fe²⁺ + NO₃⁻ + 4H⁺ → 3Fe³⁺ + NO↑ + 2H₂O
3Fe²⁺ + NO + SO₄²⁻ → [Fe(NO)]SO₄（棕色）

现象解释：
斜持试管，沿管壁缓慢加入浓H₂SO₄后，在两液面交界处出现一棕色环（生成亚硝酸合铁(Ⅱ)配合物[Fe(NO)]SO₄，特征反应，用于检验NO₃⁻）。

安全提示：
1.	防腐蚀：浓H₂SO₄具有强腐蚀性和脱水性，操作时戴手套，缓慢沿管壁加入，避免飞溅。
2.	防烫伤：浓H₂SO₄稀释时大量放热，试管外壁可能发烫，用手持试管上部或使用试管夹。
3.	防污染：实验后试管含重金属离子废液，应回收处理，勿随意倾倒。

16.第17章 一 （5）难溶性硅酸盐的生成——“水中花园”
实验原理：
Na₂SiO₃ + M²⁺ → MSiO₃↓（有色硅酸盐沉淀）
（M²⁺ = Ca²⁺、Cu²⁺、Co²⁺、Ni²⁺、Mn²⁺、Zn²⁺、Fe²⁺、Fe³⁺等）

现象解释：
通过金属盐与硅酸钠反应，生成不同颜色的金属硅酸盐胶体，在固液接触面形成半透膜，由于渗透压的关系，水不断渗入膜内，胀破半透膜使金属盐又与硅酸钠接触，又生成新的胶状金属硅酸盐。 反复渗透，反复生成出芽状或树枝状的硅酸盐，从而产生水中花园的奇妙现象。
安全提示：
1.	防腐蚀：水玻璃（Na₂SiO₃溶液）呈碱性，具有腐蚀性，避免皮肤接触。
2.	防中毒：Co(NO₃)₂、NiSO₄、MnSO₄、CuSO₄等重金属盐具有毒性，避免接触和吸入，实验后及时洗手。
3.	废液处理：含重金属离子的废液应回收处理，勿随意倾倒

17.第13章 三 1 ①KI + 浓硫酸 | 湿的醋酸铅试纸
实验一：KI与浓硫酸的反应
实验原理：
8KI + 9H₂SO₄(浓) → 4I₂ + 8KHSO₄ + H₂S↑ + 4H₂O
Pb(Ac)₂ + H₂S → PbS↓ + 2HAc
现象解释：
•	生成紫黑色固体（I₂）和臭鸡蛋气味气体（H₂S），醋酸铅试纸变黑（H₂S与Pb(Ac)₂生成PbS黑色沉淀），KI-淀粉试纸变蓝（I₂遇淀粉），pH试纸变红（生成酸性气体）。
安全提示：
1.	防中毒：反应产生H₂S（有毒）和I₂蒸气，必须在通风橱中操作。
2.	防腐蚀：浓硫酸具有强腐蚀性，避免皮肤接触。
3.	防烫伤：反应放热剧烈，试管口勿对人。

18. KBr + 浓硫酸 | 湿的KI-淀粉试纸
实验原理：
2KBr + 3H₂SO₄(浓) → Br₂ + 2KHSO₄ + SO₂↑ + 2H₂O
SO₂ + I₂ + 2H₂O → H₂SO₄ + 2HI
现象解释：
•	生成棕色物质（Br₂）和刺激性气味气体（SO₂），KI-淀粉试纸变蓝（SO₂还原I₂为I⁻，但实际反应中Br₂氧化I⁻生成I₂使试纸变蓝），pH试纸变红（SO₂溶于水生成H₂SO₃）。
安全提示：
1.	防中毒：反应产生Br₂蒸气和SO₂有毒气体，在通风橱中操作。
2.	防腐蚀：浓硫酸和Br₂均有腐蚀性，避免皮肤接触。
3.	防烫伤：反应放热，用试管夹夹持。

19. KCl + 浓硫酸 | 湿的pH试纸
实验原理：
KCl + H₂SO₄(浓) → KHSO₄ + HCl↑
现象解释：
•	产生刺激性气味气体（HCl），pH试纸变红（HCl溶于水呈酸性），醋酸铅试纸和KI-淀粉试纸无明显变化（无H₂S和氧化性气体生成）。
安全提示：
1.	防中毒：HCl气体有刺激性，在通风橱中操作。
2.	防腐蚀：浓硫酸具有腐蚀性，避免皮肤接触。

20. NaClO + MnSO₄
实验原理：
Mn²⁺ + ClO⁻ + 2OH⁻ → MnO₂↓ + Cl⁻ + H₂O
（注：NaClO溶液本身呈碱性，提供OH⁻）
现象解释：
•	溶液中迅速产生棕黑色沉淀（MnO₂），并可能伴有少量气泡（O₂，H₂O₂分解副反应）。
安全提示：
1.	防腐蚀：NaClO具有腐蚀性和漂白性，避免皮肤接触。
2.	废液处理：含重金属Mn的废液应回收处理。

21. NaClO + 品红溶液
实验原理：
NaClO + 品红 → 无色产物（氧化还原反应，破坏发色基团）
现象解释：
•	品红溶液的红色立即褪去（NaClO具有强氧化性，能破坏品红中的共轭体系，使其褪色，此为漂白性）。
安全提示：
1.	防腐蚀：NaClO具有腐蚀性和漂白性，避免皮肤接触衣物。

22. NaClO + KI-淀粉 | 酸性体系
实验原理：
2I⁻ + ClO⁻ + 2H⁺ → I₂ + Cl⁻ + H₂O
I₂ + 5ClO⁻ + H₂O → 2IO₃⁻ + 5Cl⁻ + 2H⁺（ClO⁻过量时）
现象解释：
•	溶液立即变为蓝色（I₂遇淀粉），继续滴加NaClO，蓝色褪去（I₂被过量ClO⁻进一步氧化为无色的IO₃⁻）。
安全提示：
1.	防腐蚀：NaClO和H₂SO₄具有腐蚀性，避免皮肤接触。

23. KClO₃ + 浓盐酸 | 湿 KI-淀粉试纸
实验原理：
KClO₃ + 6HCl(浓) → 3Cl₂↑ + KCl + 3H₂O
现象解释：
•	产生黄绿色气体（Cl₂），有刺激性气味，湿润的KI-淀粉试纸变蓝（Cl₂氧化I⁻为I₂）。
安全提示：
1.	防中毒：反应产生Cl₂有毒气体，必须在通风橱中操作。
2.	防腐蚀：浓盐酸具有腐蚀性，避免皮肤接触。
3.	防爆炸：KClO₃与还原剂混合加热可能爆炸，避免混入有机物。

24. KClO₃ + Na₂SO₃ + AgNO₃ 
实验原理：
ClO₃⁻ + 3SO₃²⁻ + 6H⁺ → Cl⁻ + 3SO₄²⁻ + 3H₂O
现象解释：
•	中性条件：无明显现象（反应动力学缓慢，实际观察不到反应）。
•	酸性条件：反应发生，酸化后用AgNO₃检验有白色沉淀（生成Cl⁻，与Ag⁺生成AgCl沉淀）。
安全提示：
1.	防腐蚀：H₂SO₄具有腐蚀性，避免皮肤接触。
2.	废液处理：含Ag⁺的废液应回收处理（Ag⁺为重金属，污染环境）。

25. KClO₃ + KI + CCl₄
实验原理：
6I⁻ + ClO₃⁻ + 6H⁺ → 3I₂ + Cl⁻ + 3H₂O
3I₂ + 5ClO₃⁻ + 3H₂O → 5Cl⁻ + 6IO₃⁻（ClO₃⁻过量时）
5Cl⁻ + ClO₃⁻ + 6H⁺ → 3Cl₂↑ + 3H₂O
现象解释：
•	中性条件：CCl₄层无色（ClO₃⁻氧化性较弱，不能氧化I⁻）。
•	酸性条件：CCl₄层逐渐呈现紫红色（I₂在CCl₄中的特征颜色），继续反应颜色可能变浅并产生黄绿色气体（I₂被进一步氧化为IO₃⁻，Cl⁻与ClO₃⁻反应生成Cl₂）。
安全提示：
1.	防中毒：反应可能产生Cl₂，在通风橱中操作。
2.	防腐蚀：H₂SO₄具有腐蚀性，避免皮肤接触。
3.	废液处理：含CCl₄和I₂的废液应回收处理。

26. KClO₃ + KI-淀粉 | 酸性体系

ClO₃⁻ + 6I⁻ + 6H⁺ → 3I₂ + Cl⁻ + 3H₂O

27. 高氯酸盐的氧化性
实验原理：
ClO₄⁻ + 8I⁻ + 8H⁺ → 4I₂ + Cl⁻ + 4H₂O

现象解释：
1.	KClO₄ + KI-淀粉（中性）：无明显现象（ClO₄⁻氧化性更弱）。
2.	酸化后：KClO₄能使KI-淀粉变蓝（酸性条件氧化性增强）。
3.	结论：氧化性顺序：ClO⁻（中性）> ClO₃⁻（酸性）> ClO₄⁻（酸性），酸性能显著增强ClO₃⁻和ClO₄⁻的氧化性。

1.	防腐蚀：H₂SO₄具有腐蚀性，避免皮肤接触

28.卤化银的感光性
实验原理：
AgX → Ag + X₂（X = Cl、Br、I，光照分解）
现象解释：
•	在培养皿上制备AgCl、AgBr、AgI悬浊液，部分遮光，光照一段时间后阴影轮廓清晰可见（卤化银见光分解生成黑色Ag单质，未被光照部分保持原色，形成影像），分解速率：AgCl > AgBr > AgI。

29.过氧化氢的酸性
实验原理：
2H₂O₂ → O₂↑ + 2H₂O（碱性条件催化分解）
现象解释：
•	加入NaOH和乙醇后，剧烈产生大量无色气泡（O₂），溶液迅速冒泡，可能伴有轻微放热（H₂O₂在碱性介质中不稳定，发生歧化分解）。
